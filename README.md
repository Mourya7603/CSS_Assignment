## Getting Started

### Prerequisites

No special prerequisites needed. Just a modern web browser.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Mourya7603/CSS_Assignment.git
   cd ghost-blog
   # Using Python
    python -m http.server 8000
   # Using Node.js
     npx serve
   # Using PHP
     php -S localhost:8000
   ```
